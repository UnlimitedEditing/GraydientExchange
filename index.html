Quick Start

Get Started in Seconds

Simple, intuitive API that integrates seamlessly with your existing Graydient workflow.

Parse Commands

Use Telegram-style syntax with positive /methods and negative [handles] for intuitive control.

  • ✓ /draw for image generation
  • ✓ /animate for video creation
  • ✓ [negative] for exclusions
  • ✓ --parameters for fine-tuning
from graydient_exchange import Exchange
from graydient_toolkit import (
    MethodRegistry,
    CommandParserEngine,
    InputModifier
)

# Initialize
exchange = Exchange()
registry = MethodRegistry(exchange)
parser = CommandParserEngine(registry)
modifier = InputModifier(registry)

# Parse command
parsed = parser.parse(
    "/draw a cyberpunk cat [blurry] --seed 42"
)

# Transform & Execute
result = modifier.transform(parsed)
output = exchange.run(
    result.workflow,
    result.params
)

# Example: After Effects CEP Extension
import graydient_exchange as gx
from graydient_toolkit import CommandParserEngine

# Initialize the exchange
exchange = gx.Exchange()
parser = CommandParserEngine()

# Generate from text prompt
def generateLayer(prompt, duration=5):
    parsed = parser.parse(f"/draw {prompt} --width 1920 --height 1080")
    result = exchange.run(parsed.workflow, parsed.params)
    
    # Import into After Effects
    app.project.importFile(result["image_url"])
    return "Layer created successfully!"

# User calls from extension panel
generateLayer("futuristic city skyline at dusk")

Deployment Guide

Follow these steps to integrate the toolkit into your project—whether it's a CLI tool, web app, or creative software extension.

1

Copy the Toolkit

Copy the graydient_toolkit directory to your project folder alongside graydient_exchange.

cp -r graydient_toolkit /path/to/your/project/
2

Set Up Environment

Ensure your Graydient API key is configured. The toolkit will use GRAYDIENT_KEY from environment variables.

export GRAYDIENT_KEY=your_api_key_here
# Or create a .env file with GRAYDIENT_KEY=your_api_key
3

Import and Initialize

Import the toolkit modules in your Python code and initialize the components.

from graydient_toolkit import (
    MethodRegistry,
    CommandParserEngine,
    InputModifier
)
4

Run the Example

Test your setup by running the basic usage example.

python graydient_toolkit/examples/basic_usage.py
5

Explore the CLI

Use the command-line interface to manage methods and create tutorials.

python -m graydient_toolkit.toolkit_cli --help
